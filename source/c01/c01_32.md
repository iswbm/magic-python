# 1.32 如何让大数变得更易于阅读？

![](http://image.iswbm.com/20200804124133.png)

当一个数非常大时，可能过百万，也可能上亿，太多位的数字 ，会给我们阅读带来很大的障碍。

比如下面这个数，你能一下子说出它是多少万呢，还是多少亿呢？

```
281028344
```

是不是没法很快的辩识出来？

这时候，你可以使用 `_` 来辅助标识，写成这样子就清晰多了

```
281_028_344
```

关键这种写法，在代码中并不会报错噢（Python2 不支持）

```python
>>> number=281_028_344
>>> number
281028344
```
