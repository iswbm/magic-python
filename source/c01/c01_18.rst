1.18 正负得负，负负得正
=======================

|image0|

从初中开始，我们就开始接触了\ ``负数`` ，并且都知道了\ ``负负得正``
的思想。

Python 作为一门高级语言，它的编写符合人类的思维逻辑，包括 ``负负得正``
。

.. code:: python

   >>> 5-3
   2
   >>> 5--3
   8
   >>> 5+-3
   2
   >>> 5++3
   8
   >>> 5---3
   2
