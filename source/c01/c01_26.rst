1.26 x 与 +x 等价吗？
=====================

.. image:: http://image.iswbm.com/20200804124133.png

在大多数情况下，这个等式是成立的。

.. code:: python

   >>> n1 = 10086
   >>> n2 = +n1
   >>>
   >>> n1 == n2
   True

什么情况下，这个等式会不成立呢？

由于Counter的机制，\ ``+`` 用于两个 Counter
实例相加，而相加的结果如果元素的个数 ``<=`` 0，就会被丢弃。

.. code:: python

   >>> from collections import Counter
   >>> ct = Counter('abcdbcaa')
   >>> ct
   Counter({'a': 3, 'b': 2, 'c': 2, 'd': 1})
   >>> ct['c'] = 0
   >>> ct['d'] = -2
   >>>
   >>> ct
   Counter({'a': 3, 'b': 2, 'c': 0, 'd': -2})
   >>>
   >>> +ct
   Counter({'a': 3, 'b': 2})
