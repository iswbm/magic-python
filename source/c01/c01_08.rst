1.8 连接多个列表最极客的方式
============================

|image0|

.. code:: python

   >>> a = [1,2]
   >>> b = [3,4]
   >>> c = [5,6]
   >>>
   >>> sum((a,b,c), [])
   [1, 2, 3, 4, 5, 6]
