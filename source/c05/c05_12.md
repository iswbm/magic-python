# 5.12 逗号也有它的独特用法
![](https://image.iswbm.com/20200804124133.png)

逗号，虽然是个很不起眼的符号，但在 Python 中也有他的用武之地。

**第一个用法**

元组的转化

```shell
[root@localhost ~]# cat demo.py 
def func():
    return "ok",

print(func())
[root@localhost ~]# python3 demo.py 
('ok',)
```

**第二个用法**

print 的取消换行

```shell
[root@localhost ~]# cat demo.py 
for i in range(3):
    print i
[root@localhost ~]# 
[root@localhost ~]# python demo.py 
0
1
2
[root@localhost ~]# 
[root@localhost ~]# vim demo.py 
[root@localhost ~]# 
[root@localhost ~]# cat demo.py 
for i in range(3):
    print i,
[root@localhost ~]# 
[root@localhost ~]# python demo.py 
0 1 2
[root@localhost ~]#
```
